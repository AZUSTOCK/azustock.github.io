# 補習
<div class="vertical-wrapper">

「戶賀同學你好，最近過的如何？ 」  
「誒。還...還不錯。」  
「是嗎？ 那太好了。 戶賀同學最近常常請假，老師有點擔心，老師還以為自己被討厭了呢QAQ。」  
老師還是一如既往地愛開玩笑呢。  
「抱歉，讓老師擔心了，不過我完全沒事，只是最近學校事情有點多。」  
「這樣啊，老師了解了。不過啊，錄的影片有好好看完嗎？」  
「有的，全部都看完了。」  
「是，老師知道了。辛苦了~」  
<br/>
本來想請假的，不過出席一下比較好吧。  
<br/>  
「所以妳今天不能出去玩！」  
在上最後一堂課前，我跟夜中同學說了老師打了電話的事。  
「都被老師關切了，感覺再請假過意不去。」  
「欸，真無聊。」  
隨後夜中同學走回了座位，她看起來相當失落。  
<br/>
看起來這個偷來的時光要結束了。  
也不是說我不喜歡補習，補習班老師人很好，課教的淺顯易懂，還很會講笑話，每次去都很開心。但是我可是高中生！現在正是綻放的時候。等到畢業後，凋謝變成了種子，就只能待在土壤裡，哪都不能去了。  
因此每當坐在電車裡看著窗外，在店裡結伴的學生們，總是會有種羨慕感浮現。  
<br/>
『如果不是夜中同學的話，我會怎麼樣呢？』  
我看著在座位上的夜中同學，思考著這幾天發生的事情，雖然每個行程都不長，而且很亂來，但是回憶中的我看起來卻很開心。  
<br/>
『我應該就是默默地過完高中生活吧。』  
我不是班上的中心人物，也不是被排擠的對象，單單只是普通的同學而已。因為是轉學過來的，所以大家的小圈圈都建好了，我也不好意思去破壞他們這一年來建立的平衡，於是就平淡地過著我的高中生活。遇到了會打聲招呼，有時候會一起聊天說笑，但是放學後的記憶裡，從來沒有同學的身影。  
<br/>  
我看向了空蕩蕩的座位  
『沒有。』  
站了起來跑到了走廊上  
『不在。』  
我跑下了樓梯，往校門口的方向跑，  
『等一下！』  
站到了夜中同學的前面，她的眼神帶了點疑惑。  
我想我也是，我不知道為什麼我有這樣的想法，我也不知道這樣行不行得通，但是我還是說出口了。  
「拜託了，能不能幫我補習？」  
「誒？」  
這是我第一次看到她不知所措的樣子。
